# 手机测评中心 - GitHub Pages

这是一个静态版本的展示页面。

## 完整功能版本

完整的PHP版本包含数据库和动态功能，请访问：
- GitHub仓库: https://github.com/hinoishicn/phone-review-center
- 完整功能演示: 需要PHP服务器环境

## 功能特色

- ✅ 50+手机型号数据
- ✅ 详细规格参数对比
- ✅ 智能搜索筛选
- ✅ PWA支持
- ✅ Android应用
- ✅ 响应式设计

## 技术栈

- **前端**: HTML5, CSS3, Bootstrap 5, JavaScript
- **后端**: PHP 7.4+, MySQL 5.4+
- **移动端**: PWA, Android WebView
- **部署**: GitHub Pages, GitHub Actions
