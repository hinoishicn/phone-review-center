# itops-help-center

ITOps 帮助中心是一个提供IT运维相关指南、工具和支持的平台，旨在帮助运维人员快速解决日常工作中的问题。

## 项目结构
- `index.php`：网站首页，包含导航链接
- `version.php`：版本记录页面，展示项目更新历史
- `README.md`：项目说明文档

## 本地运行
1. 将项目克隆到本地：`git clone https://github.com/hinoishicn/itops-help-center.git`
2. 确保本地安装了PHP环境（如XAMPP、WAMP或单独的PHP解释器）
3. 将项目放入服务器根目录（如XAMPP的`htdocs`文件夹）
4. 通过浏览器访问：`http://localhost/itops-help-center/index.php`

## 版本记录
详见网站内的「版本记录」页面，或直接访问 `version.php`。
